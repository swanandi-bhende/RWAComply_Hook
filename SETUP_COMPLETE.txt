🔍 TROUBLESHOOTING - IF SOMETHING ISN'T WORKING
================================================================================

❌ Can't connect wallet
   → Refresh page (Cmd+Shift+R)
   → Make sure MetaMask is installed
   → Check MetaMask has Anvil network configured
   → Try clicking Connect again

❌ Balances show 0.0000
   → Make sure you're on Account 1 (has 1000 ETH by default)
   → Verify MetaMask network is "Anvil" (31337)
   → Hard refresh: Cmd+Shift+R

❌ Oracle values show "N/A" or "Loading..."
   → Wait 2-3 seconds (they're fetching from contracts)
   → Hard refresh page
   → Check Anvil is running: curl http://127.0.0.1:8545
   → Check no errors in browser console (F12)

❌ Admin tab not visible
   → Make SURE you're on Account 1 with address:
      0xf39Fd6e51aad88F6F4ce6aB8827279cffFb92266
   → Connected wallet? Check top-right
   → On correct network (31337)? Check MetaMask
   → Try refreshing page

❌ Frontend not loading at http://localhost:3000
   → Dev server crashed? Check terminal
   → Restart: cd frontend && npm run dev
   → Wait 5-10 seconds for it to rebuild
   → Try again

❌ Need to restart everything
